AWS and DevOps Components

Below is a simple overview of common AWS and DevOps components.

AWS Components

  • EC2 (Compute)
  • S3 (Storage)
  • RDS / Aurora (Databases)
  • Lambda (Serverless)
  • VPC (Networking)
  • IAM (Identity & Access)
  • CloudFormation / CDK (IaC)
  • CloudWatch (Monitoring)
  • EKS / ECS (Containers)

DevOps Components

  • Source Control: Git / GitHub / GitLab
  • CI/CD: Jenkins / GitHub Actions / GitLab CI
  • Containers: Docker
  • Orchestration: Kubernetes
  • IaC: Terraform / Ansible
  • Config Mgmt: Chef / Puppet / Ansible
  • Monitoring: Prometheus / Grafana
  • Artifact Repo: Nexus / Artifactory
